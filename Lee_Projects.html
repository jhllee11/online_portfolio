Dinosaur Parkour


Course: Computer Science Principles
Game: Dinosaur Parkour
Duration: One week (September 3 to September 11)
Game Documentation

Description:

In this project, I worked with a partner to create and develop a video game app. The objective of this game is to control a character and try to reach the lightning bolts without encountering the obstacles(dinosaurs) in both levels. This was the first project of the year in the Computer Science Principles course and it introduced the world of coding and programming through building blocks. As we programmed this game, we learned about the complexity of the functions in the codes used for the features of the game. In addition, the game we made was original, so the process and thought to build a new game with different features and abilities took a long time, but overall this project was a good learning experience on how to work and cooperate with other peers and how to use other sources given effectively to add more development within the codes.
